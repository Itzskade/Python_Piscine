## Mastering the Mysteries of Python Imports

This repository contains all exercises for **The Codex: Alchemical Python Imports**.  
The project guides you through **the four sacred mysteries of Python imports**: package initialization, import paths, absolute vs. relative imports, and breaking circular dependencies.

As a Python Alchemist, you will **organize scattered code into magical packages**, invoke specific spells (functions) efficiently, and avoid the traps of circular dependencies.

## ğŸ“ Rules and Best Practices

- Python **3.10+** required  
- Only standard library modules allowed  
- No `eval()`, `exec()`, or external packages  
- Keep functions simple, returning **strings**  
- Apply **type hints** consistently  
- Handle errors with `try/except`  
- Use `__init__.py` to expose only intended functions  
- Demonstrate understanding of absolute vs relative imports  
- Avoid circular dependencies; if needed, use deferred imports or dependency injection
