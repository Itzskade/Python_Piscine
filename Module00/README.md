## Python Fundamentals Through Community Garden Data

This repository contains all the exercises for the **Cultivating Code** project, an introductory project designed to teach the fundamentals of **Python 3** using realistic scenarios based on community gardens.  
The project covers essential concepts such as standard input/output, variables, conditionals, loops, recursion, and basic type annotations.

- Apply basic type annotations

## ğŸ“ Rules and Style Guidelines
